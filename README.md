# 🍰 نظام إدارة مبيعات البسبوسة

نظام متكامل لإدارة مشروع البسبوسة المنزلية — تتبع المكونات، المشتريات، الوصفات، الإنتاج، المبيعات، والأرباح.

---

## ✨ المميزات

| الوحدة | الوصف |
|--------|-------|
| 📦 **المكونات** | إضافة وتتبع المكونات الغذائية ومستلزمات التغليف مع تنبيهات المخزون المنخفض |
| 🛒 **المشتريات** | تسجيل كل عملية شراء وتحديث المخزون تلقائياً |
| 📋 **الوصفات** | إنشاء وصفات متعددة مع حساب تكلفة الصينية وتكلفة القطعة |
| 🏭 **الإنتاج** | تسجيل الإنتاج اليومي مع خصم المكونات من المخزون تلقائياً |
| 💰 **المبيعات** | تسجيل المبيعات مرتبطة بكل وصفة مع حساب الربح لكل عملية |
| 📊 **التقارير** | تقارير تفصيلية للإيرادات والتكاليف والأرباح لأي فترة زمنية |
| 🖥️ **لوحة التحكم** | ملخص يومي وشهري مع تنبيهات المخزون المنخفض |

---

## 🚀 تشغيل المشروع

### المتطلبات
- [Node.js](https://nodejs.org/) مثبّت على الجهاز

### الخطوات

```bash
# 1. انتقل إلى مجلد المشروع
cd my-business

# 2. شغّل السيرفر
node server.js

# 3. افتح المتصفح على
http://localhost:3000
```

---

## 🗂️ هيكل المشروع

```
my-business/
├── index.html        # واجهة التطبيق الكاملة
├── server.js         # سيرفر Node.js بسيط
├── css/
│   └── styles.css    # تنسيقات عربية RTL
└── js/
    ├── db.js         # طبقة البيانات (localStorage)
    └── app.js        # منطق الواجهة والوحدات
```

---

## 🛠️ التقنيات المستخدمة

- **HTML / CSS / JavaScript** — بدون أي مكتبات خارجية
- **localStorage** — لحفظ البيانات محلياً في المتصفح
- **Node.js** — لتشغيل سيرفر محلي بسيط
- **RTL** — واجهة عربية كاملة من اليمين لليسار
- **العملة** — درهم إماراتي (د.إ)

---

## 📌 ملاحظات

- البيانات محفوظة في المتصفح (`localStorage`) — لا تُمسح إلا إذا مسحت بيانات المتصفح يدوياً
- عند تسجيل **إنتاج جديد** تُخصم كميات المكونات من المخزون تلقائياً
- عند **حذف إنتاج** تُعاد الكميات للمخزون تلقائياً
- يمكن تحديد **حد أدنى للمخزون** لكل مكوّن لتفعيل تنبيهات لوحة التحكم
